
## Journal App

### Technologies Used:
- Node.js
- Postgres
- Express.js
- Sequelize
- React
- Redux
- Javascript
- Create-recat-app package

### Installation:
- Have postgres installed (https://postgresapp.com/)
- Create a database called journal
- npm install
- cd client
- npm install 
- cd ..
- npm run seed
- npm run client
- npm run server (http://localhost:8000)
- npm test
